# 前端开发实施计划

根据 PRD 和技术架构文档，我们将构建基于 **Vue 3 + TypeScript + Vant UI** 的移动端 H5 前端应用。

## 1. 项目初始化 (Project Setup)
- [ ] **创建项目**: 在 `frontend` 目录下初始化 Vite + Vue 3 + TypeScript 项目。
- [ ] **安装依赖**:
    -   **UI 框架**: `vant` (移动端组件库)
    -   **路由**: `vue-router`
    -   **状态管理**: `pinia`
    -   **HTTP 客户端**: `axios`
    -   **工具**: `vuedraggable` (用于拖拽排序), `sass` (CSS 预处理)

## 2. 基础架构搭建 (Infrastructure)
- [ ] **目录结构**:
    -   `src/api`: API 接口定义 (包含 Mock 数据)
    -   `src/views`: 页面组件
    -   `src/components`: 通用组件
    -   `src/stores`: 状态管理 (ProjectStore)
- [ ] **Vant 配置**: 全局引入或按需引入配置，适配移动端 Viewport。
- [ ] **路由配置**: 定义 `/create`, `/review`, `/result` 等路由。

## 3. 核心页面实现 (Page Implementation)

### 3.1 新建项目与录入页 (Create & Input)
> 对应 PRD 5.1
- [ ] **基础信息表单**: 小区名称、户型结构(Picker)、建筑面积、挂牌价格。
- [ ] **核心卖点选择**: Checkbox 组 (学区房、地铁沿线等)。
- [ ] **素材上传组件**: 使用 Vant Uploader 实现多视频上传 UI (模拟上传进度)。

### 3.2 智能分段确认页 (Smart Review)
> 对应 PRD 5.2
- [ ] **时间轴列表**: 展示视频片段缩略图、时长、AI 识别标签。
- [ ] **拖拽排序**: 集成 `vuedraggable` 实现卡片上下拖拽调整顺序。
- [ ] **标签编辑**: 点击标签弹出 Vant Picker 进行修改。
- [ ] **解说词预览**: 底部展示生成的脚本内容，支持简单的文本编辑。

### 3.3 视频预览与结果页 (Result)
> 对应 PRD 5.3 & Phase 5.3
- [ ] **视频播放器**: 展示最终合成的视频。
- [ ] **操作栏**: 下载视频、复制链接、重新生成按钮。

## 4. Mock 数据与联调 (Mocking)
- [ ] **Mock Service**: 创建简单的 Mock 服务模拟后端 API (`/api/v1/projects` 等)，确保前端流程可以跑通 (上传 -> 模拟分析延时 -> 确认页 -> 模拟渲染延时 -> 结果页)。

## 执行顺序
我们将按顺序执行：初始化 -> 基础架构 -> 页面开发 (录入 -> 确认 -> 结果) -> Mock 联调。
